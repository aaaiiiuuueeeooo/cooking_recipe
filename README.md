# サイト名
お手軽！クッキングレシピ

## サイト概要
### サイトテーマ
この料理を作りたいとなった時に手軽にレシピを探せるコミュニティサイト

### テーマを選んだ理由
料理をするにあたってレシピを見ながら作ることも多いと思うので手軽にレシピを見ながら作れたら良いなと思いこのテーマにしました

### ターゲットユーザ
料理が得意な人、苦手な人、好きな人

### 主な利用シーン
料理を作る際に手軽にレシピを探したりできる、自分のレシピを投稿したりできる
